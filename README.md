# Homebrew Tap

Custom Homebrew formulas by [@karbassi](https://github.com/karbassi).

## Installation

```bash
brew tap karbassi/tap
```

## Formulas

| Formula | Description |
|---------|-------------|
| [cc-statusline](https://github.com/karbassi/cc-status-line) | Lightweight, fast status line for Claude Code CLI |

### cc-statusline

```bash
brew install karbassi/tap/cc-statusline
```
